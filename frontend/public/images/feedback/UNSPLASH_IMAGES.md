# Feedback Images

The feedback category cards use high-quality Unsplash images loaded through `https://images.unsplash.com/...` URLs in `frontend/src/app/pages/home.component.html` and `frontend/src/main.css`.

Categories covered:

- Faculty / campus building
- Teachers / classroom
- Courses / study group
- Transport / bus
- Cafeteria / restaurant
- Sports / football/field
- Library / books and reading area
- Events / campus event

Note: Because these are externally hosted Unsplash images, an internet connection is required for the photos to load. The layout still has a gradient fallback if an image cannot load.
